Steps for configuring the software are:
1) Create a Diretory in your home (or anywhere else) called LAXPCsoftware
2) untar the tar file 'laxpcsoft.tar.gz' in the LAXPCsoftware directory e.g. /home/yourname/LAXPCsoftware/
3) Put the following commands in the .bashrc file located at you home directory
export LAXPCSOFT="/home/yourname/LAXPCsoftware/"
export LAXPCPATH="$LAXPCSOFT/LAXPC_CAL/"
export PATH="$PATH:$LAXPCSOFT/laxpc_bin"
4) Open a new terminal
5) In LAXPCsoftware run the command
./Configure
Note if there are any errors

---------------------------------------------------------------------------
Analyzing data
1) In a directory where you want to keep the results of the analysis e.g. myresults_cygx1
2) Set the variable $LAXPCDATAPATH to where your data is ending with the laxpc directory..e.g.
export LAXPCDATAPATH="/home/rmisra/Cyg_X1_new.dir/april2016/20160601_G05_191T01_9000000476_level1/laxpc/"
3) Make data file lists by giving the command
laxpc_make_filelist
this will produce two text files "eventfiles" and "filterfiles"
4) Make FITS event file by giving the command laxpc_make_event eventfiles
this will produce a file called 'level2.event.fits'.
5) Make gti file laxpc_make_stdgti filterfiles
which will produce a file called 'usergti' which has the GOOD TIME INTERVALS (GTI)which removes earth occulation and SAA. (Note: "laxpc_make_occultgti filterfiles" will instead make occultgti having the time when the source was occulted).
6) Now you can use the following codes to make lightcurve, spectra, power spectra and time-lags with flags defined..
(A) To Make Lightcurves: laxpc_make_lightcurve [see README_LC]
(B) To Make Spectra: laxpc_make_spectra [see README_SPEC]
(C) To Make background spectra: laxpc_find_back [see README_BACKSPEC]
(D) To Make background lightcurves: laxpc_make_backlightcurve [see README_BACKLC]
(E) To Make Power spectra, Time Lags as a function of energy and frequeny: laxpc_find_freqlag [see README_TIMELAG]
(F) To rebin power spectra and make them readable by Xspec: laxpc_rebin_power [see README_rebin_power]
(G) To Make Dynamic power spectra laxpc_dynpower [see README_DYNPOW]
(H) To Make Dynamic power spectra laxpc_fluxresl_gti [see README_FLUXRESLGTI]
(I) To create background and lightcurve using an alternate scheme for faint sources laxpc_backspectra_F_L20, laxpc_backlc_F_L20, laxpc_backspectra_F_L10, laxpc_backlc_F_L10 [see README_BACKSPEC_F_L10 README_BACKSPEC_F_L20 README_BACKLCF_L10 README_BACKLCF_L20 ]
(J) To create xspec readable pha files for rms and phase lag as a function of energy:
laxpc_make_lagrms_pha [See README_Make_Lagrms_pha]
