Programme to find background lightcurve for Faint sources (see report) for LAXPC 10

Sevarel steps have to be done to run this programma

1) obtain a lightcurve in the 50 80 keV range for LAXPC 10 Layer 1 only

laxpc_make_lightcurve -t 100. -u usergti -l 1 -p 1 -e eneinp level2.event.fits

where eneinp is a file containing the following
50.0 80.0

2) Select only that GTI where the flux is within 14 and 24 c/s

laxpc_fluxresl_gti -l 14.0 -h 24.0 -o usergtia lightcurve_L1_50.0_80.0keV.lc

3) Make lightcurve in 50-80 keV using the new gti file usergtia

laxpc_make_lightcurve -t 100. -u usergtia -l 1 -p 1 -e eneinp level2.event.fits

4) Create lightcurve using new GTI file

laxpc_make_lightcurve -t XX -u usergtia -l 1 -p 1 -e eneyour level2.event.fits

where XX is the time bin for which you want to create lightcurve and eneyour is the file having energy ranges that you want but MUST NOT have any enegy range above 30 keV.

5) Create background lighcturve:

laxpc_backlc_F_L10 -e eneyour lightcurve_L1_50.0_80.0keV.lc

This will creat a background lightcurves: Back_lightcurve_L10_F_**_**keV.lc
where ** are the energy ranges specfied in the eneyour file.

Note that the background lightcurve will always be of 100 second time bin
